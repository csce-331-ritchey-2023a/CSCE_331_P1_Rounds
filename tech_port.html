Project 1: Seat Swapper
One of my technical projects was from Texas A&M's, TAMU Hack 2023, my group and I worked together to create a website that connect individuals on specific flights to allow the to trade seats with one another. Consider your on a flight and you really hate the window seat and you ask the people in your row but they are not willing to switch seats, it is likely that there is someone in a similar situation (except they are in an aisle seat and would prefer the window)in a different row. Fortunately this common issue is solved by Seat Swapper which allows users to log onto the website and select their seat and toggle yes or no depending on if they are willing to swap. This data is public so there users can then check and see if someone in their dream seat has toggled yes to switch. From there, with in website communication, users can conveene and swap.

    Another technical project was an individual course project that implemented a red black using a linked list. My completed project includes node insertion, deletion, and a search function. In the event of node insertion or deletion the tree would rebalence if need with a left or right shift.
